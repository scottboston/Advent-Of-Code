# Advent of Code Solutions

This repository contains my solutions for the [Advent of Code](https://adventofcode.com/) challenges.

## Structure

The project is organized by year, with each directory containing the solutions for that specific year's events.

- `2015/`
- `2016/`
- `2019/`
- `2022/`
- `2023/`
- `2024/`
- `2025/` - Current year in progress

## Environment Setup

This project uses **Python 3.13.3** and manages dependencies with `uv`.

### Prerequisites

- Python 3.13+
- [uv](https://github.com/astral-sh/uv) package manager

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd Advent-Of-Code
   ```

2. Install dependencies:
   ```bash
   uv sync
   ```
